## 📖 ENGLISH VERSION

### 🎯 About The Project

**Leather Shop** is a modern and elegant online store for high-quality, handmade leather products. This project is built with **pure HTML and CSS** – no JavaScript, no frameworks, just clean and beautiful code.

### ✨ Features

| Category | Description |
|----------|-------------|
| 🛍️ **Product Categories** | Bags, Belts, Wallets, Accessories |
| 👔 **Men's Collection** | Leather bags & belts for men |
| 👛 **Women's Collection** | Handbags & accessories for women |
| 📱 **Fully Responsive** | Works perfectly on mobile, tablet & desktop |
| 🎨 **Modern Design** | Elegant leather-themed user interface |
| 📞 **Contact Form** | Easy way for customers to reach out |

### 🛠️ Technologies Used

| Technology | Purpose |
|------------|---------|
| **HTML5** | Structure and framework of the website |
| **CSS3** | Styling, animations, and responsive design |
| **Flexbox & Grid** | Modern and flexible layouts |
| **Google Fonts** | Vazirmatn (Persian) & Playfair Display (English) |
