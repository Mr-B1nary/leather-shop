پروژه فروشگاه اینترنتی چرم لوکس
==============================

نام پروژه: Leather Luxury - فروشگاه آنلاین محصولات چرمی
تکنولوژی: HTML5 و CSS3 خالص (بدون JavaScript)
تاریخ توسعه: ۱۴۰۳
توسعه‌دهنده: [نام شما]

ساختار پروژه
------------
leather-shop/
├── index.html              # صفحه اصلی
├── products.html           # صفحه محصولات
├── product-detail.html     # صفحه جزئیات محصول
├── contact.html            # صفحه تماس با ما
├── css/                    # پوشه استایل‌ها
│   ├── variables.css       # متغیرهای CSS
│   ├── style.css           # استایل اصلی
│   ├── responsive.css      # استایل واکنشگرا
│   ├── products.css        # استایل صفحه محصولات
│   ├── product-detail.css  # استایل جزئیات محصول
│   └── contact.css         # استایل صفحه تماس
├── images/                 # پوشه تصاویر
└── README.txt              # این فایل

ویژگی‌های کلیدی
---------------
1. طراحی کاملاً واکنشگرا (Responsive Design)
2. استفاده از HTML5 Semantic Elements
3. دسترسی‌پذیری (Accessibility) مناسب
4. کدنویسی تمیز و سازمان‌یافته
5. سرعت بارگذاری بالا
6. سئو بهینه شده

صفحه‌های اصلی
-------------
1. صفحه اصلی: نمایش محصولات ویژه، دسته‌بندی‌ها و ویژگی‌های فروشگاه
2. صفحه محصولات: لیست تمام محصولات با فیلتر و مرتب‌سازی
3. صفحه جزئیات محصول: نمایش کامل اطلاعات محصول
4. صفحه تماس: فرم تماس، اطلاعات و سوالات متداول

رنگ‌بندی
--------
- رنگ اصلی: #8B4513 (قهوه‌ای چرم)
- رنگ ثانویه: #D4A76A (طلایی/بژ)
- رنگ تاریک: #1A1A1A
- رنگ روشن: #F5F5F5

فونت‌ها
-------
- فونت اصلی: Vazirmatn (برای متن فارسی)
- فونت ثانویه: Playfair Display (برای تیترها)

نکات طراحی
----------
1. استفاده از Flexbox و Grid برای چیدمان
2. انیمیشن‌های ملایم برای تجربه کاربری بهتر
3. سایه‌های مناسب برای عمق بخشیدن به المان‌ها
4. طراحی مدرن و متناسب با موضوع چرم

نکات فنی
--------
1. تمامی تصاویر دارای alt text می‌باشند
2. استفاده از meta tags برای سئو
3. ساختار کد به صورت کامنت‌گذاری شده
4. رعایت اصول semantic HTML

نکات برای مسابقه
----------------
1. این پروژه با HTML و CSS خالص توسعه یافته است
2. امکان افزودن JavaScript برای تعامل بیشتر وجود دارد
3. طراحی به گونه‌ای است که برای مسابقات سطح استانی مناسب می‌باشد
4. قابلیت توسعه و افزودن صفحات بیشتر را دارد

راهنمای اجرا
------------
فقط کافی است فایل index.html را در مرورگر باز کنید.

توسعه بیشتر
-----------
امکان اضافه کردن موارد زیر وجود دارد:
1. صفحه درباره ما
2. صفحه گالری تصاویر
3. صفحه وبلاگ
4. سیستم سبد خرید (با JavaScript)
5. سیستم ورود کاربران

تماس
----
برای اطلاعات بیشتر با توسعه‌دهنده تماس بگیرید.
